Privacy Policy

This policy applies to Identity Wallet.

Data Collection

Identity Wallet does not send selected photo content, recognized text, contact records, or location coordinates to a developer backend in the current local-first MVP.

On-Device Sources

If you select photos or grant optional access, the app may analyze selected photo content, current location availability, and contact metadata counts on the device to create Open Wallet draft suggestions. Selected-photo analysis can detect text and broad visual categories. Raw photo content, raw recognized text, raw image labels, contact names, and location coordinates are not added to the Open Wallet profile. Before applying a draft, you can include or remove each suggestion and edit the visible identity line. Only included suggestions and the identity line you apply may become part of your Open Wallet.

On-Device Identity Line Polish

Identity Line Polish is optional. When Apple's on-device system language model is available, the app uses a bounded subset of the public Open Wallet you are editing — role, city, non-diagnostic signals, work, public interests, and public Questions and answers — to draft one identity line. During this step, the input and generated draft are not sent to a developer server. If the model is unavailable or the draft fails the app's checks, the app uses a deterministic local fallback. The draft is shown for review and is not applied to Open Wallet until you choose Apply Line.

No Account or Tracking

The current app does not require an account, does not include third-party analytics, and does not use data for tracking.

Sharing

When you use the QR/share flow, the public Open Wallet payload and preview fields are included in a static shared link. If you share one selected Question, its label, prompt, and public answer are also included so the receiver opens the same conversation context. Closed Wallet content and local source-evidence summaries are excluded. Anyone who receives the link can keep its public contents, and a shared static link cannot be revoked by the app.

The optional web preview is hosted on GitHub Pages. Open Wallet fields are placed in the URL fragment so browsers do not send those fields in the HTTP request to the host. GitHub may still process ordinary web-request metadata, such as IP address and browser information, under its own service policies.

Storage and Controls

Your profile and locally kept wallets remain on the device until you edit or remove them, reset the app data, or delete the app. You can change Photos, Contacts, Location, and Camera permissions in iOS Settings. Removing permission stops future access but does not automatically remove public text or tags that you previously applied to the Open Wallet.

Contact
