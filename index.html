Identity Wallet

Identity Wallet is a local-first iPhone app for creating, editing, and sharing an Open Wallet profile.

Help

If you need support, report a problem or request help through the public issue tracker.

Open Wallet Links

Shared Open Wallet links can open a lightweight browser preview. The preview shows only the Open Wallet fields included in the shared link.

Privacy

The current app is designed as a local-first MVP. It does not require an account and does not collect data on a developer server.
